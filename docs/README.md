# PresentCoach: Google credentials and environment configuration


**Present Coach** is a web app that helps users improve their presentation skills by analyzing recorded or uploaded videos.

At a high level, it:

* **Takes video input** (recorded live or uploaded)
* **Splits it into segments**
* Uses **AI (Gemini/Vertex)** to analyze each segment
* Provides **feedback on delivery**, such as clarity, pacing, and overall presentation quality

The goal is to give users **actionable, automated coaching** so they can refine how they present without needing a human evaluator.
